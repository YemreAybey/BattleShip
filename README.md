## Getting started

To get started with the app, clone the repo

```
git clone => The SSH key or URL <=
```

Change to Directory

```
cd  Battleship
```

Run

```
npm install
```

Run

```
npm test
```

You will see the results in your terminal. Then you can open the 'index.htm' file on your browser and enjoy the game.

# BattleShip

Implementation of battleship game with javascript

# Used Stacks

- Javascript
- Webpack
- Jest

#Prerequisites

- NPM
- WEBPACK
- GIT

## Getting started

To get started with the app, clone the repo

```
git clone => The SSH key or URL <=
```

Change to Directory

```
cd  testin-jest
```

Run

```
npm install
```

Run

```
npm test
```

You will see the results in your terminal.
